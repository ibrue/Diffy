Training Fidelity Breakdown

EgoCodec separates canonical (reference) cycles from repeat cycles. Canonical frames are always stored at full quality. Repeat cycles are handled according to the selected fidelity mode.

Canonical cycles — — full quality, always
Repeat cycles (clone mode) — — pixel data discarded, 4 B reference pointer
Repeat cycles (delta mode) — — variation preserved for training (~15% of canonical)
Warning — Clone Mode Data Quality

Clone mode discards per-cycle variation. Models trained on cloned data see N copies of the same canonical frame sequence. This can cause overfitting to a single reference trajectory and suppress learning of natural variation. Use Delta mode if cycles vary meaningfully (lighting, pose, part placement). Use Full mode for highest-fidelity training datasets where storage cost is acceptable.
